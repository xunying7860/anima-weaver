# Anima Weaver 更新日志

## 2026-07-09 批量管线大重构

### 新增节点
- **批量种子** (`BatchSeedNode`) — 生成 N 个种子，以多行 STRING 输出
- **同步串行** (`SyncPassthrough`) — 7路输入合并为 JSON 多行输出，支持行选择
  - 参数：`start_index`, `max_rows`, `remove_empty_lines`
  - 输出格式：每行一个 JSON 对象 `{"种子":"42","提示词":"...","画师":"@...","分辨率":"800x600","反推":"...","宽度":"800","高度":"600"}`
- **串行拆分** (`PassthroughSplit`) — 接收 JSON 行，拆回 7 路（种子 INT/提示词 STR/画师 STR/分辨率 STR/反推 STR/宽度 INT/高度 INT）
- **STRING转INT** (`StringToInt`) — 多行 STRING 取第一行转 INT

### 改造节点
- **提示词编织器** (`AnimaWeaver`)
  - 批量模式：接收 `种子串`，每种子独立 Raffle + 冲突检查 + NL 生成
  - LLM 生命周期：加载一次跑 N 组，跑完后按需卸载
  - 输出新增 4 路：`提示词串` / `画师串` / `分辨率串` / `反推串`
  - 互斥规则：`种子串` ↔ `随机种子`（批量优先），`分辨率串` ↔ `分辨率`（批量优先）
  - 冲突检查：配对互斥 + 裸体+服装检查 + 手动标签保护 + 去重 + 最终去重
  - `画幅比例` 统一改名为 `分辨率`，输出不含分辨率信息
  - LLM 提示词更新：`Aspect ratio:` → `Resolution:` + 规则 9 禁止输出分辨率
- **画师Seed** (`ArtistSeedNode`)
  - 批量接收 `种子串`，每个种子以自己的值做 random seed 独立抽画师
- **随机分辨率选择器** (`RandomResolution`)
  - 批量接收 `种子串`，每种子生成独立分辨率
  - 新增输出：`宽度串` / `高度串`
  - 输出名 `画幅比例` → `分辨率`，格式 `"Aspect Ratio: 16:9"` → `"1024x768"`
- **图片反推描述** (`ImageCaption`)
  - 提示词选择逻辑：自定义 > 有图→反推 > 无图→润色
  - 批量模式：接 `种子串`，每种子独立生成描述
  - 新增输出：`提示词串` / `画师串` / `分辨率串` / `反推串`（透传）
  - 新增输入：`待润色文本`
  - 接入图像时传给 LLM（VL 模型 base64 支持）
  - `画幅比例` → `分辨率`
  - 系统提示词加入 `目标分辨率` 和 `不要包含分辨率信息`
- **LM Studio 模块** (`lm_studio.py`)
  - 新增 `image_b64` 参数，支持 VL 模型传图
  - 提示词 `Aspect ratio:` → `Resolution:`
  - 规则 9：禁止输出分辨率信息

### 架构变更
- 批量管线：批量种子(N个) → 各节点 → 同步串行(合并JSON) → 提示词行(预览N格) → 串行拆分(拆回7路) → 下游
- 同步串行输出 1 路多行 JSON，由提示词行拆为 N 格逐行执行
- 串行拆分直接输出 INT（种子/宽度/高度），无需额外转换

### 修复
- 同步串行 `min(counts)` → `max(counts)`：修复未接输入导致全空的问题
- 分辨率串每行不再重复输出 `"744x1328 744x1328"`（删除了 `{w}x{h} {ratio}` 中的重复部分）
- 反推节点：图像正确传给 VL 模型（之前仅传文字）

### 移除
- 旧名兼容 `SyncPassthrough → SyncPassthroughList` 已移除，仅保留 `SyncPassthrough`

## 2026-07-14 并发批量 + 画师语义变更 + 云端适配

### 新增功能
- **批量 LLM 并发** — 提示词编织器和反推节点新增 `并发数` 参数（默认 4，最大 128）
  - 先加载一次模型，然后 N 个线程并行发 HTTP 请求
  - 云端 API 同样适用并发（注意 Rate Limit）
- **提示词编织器新增 `NL 系统提示词` 可选输入端口**（左上角 forceInput）
  - 逻辑同反推节点：有输入 → 覆盖默认系统提示词；未接 → 用默认的
- **批量种子最大值 256 → 4096**

### 画师种子语义变更（-1/0/正数）
- `-1` = 随机画师
- `0` = 无画师（不插入）
- `≥1` = 固定画师序号
- 主节点同步适配新语义

### 功能改进
- 反推节点和主节点的 `API地址` 从下拉选择改为可自由输入的文本字段
- 英文 README（`README.md`）、中文说明文档（`README_CN.md`）
- API 密钥字段默认不公开，但兼容旧版 ComfyUI（未使用 PASSWORD 类型）
- `enable_thinking` 参数仅在本地 LM Studio 时发送，云端 API 跳过（修复 SiliconFlow 400 错误）
- LM Studio 路径搜索增加 `.lmstudio/bin/` 位置

### 提示词系统
- 反推节点系统提示词全部改为英文（`_build_system_prompt` / `_build_batch_prompt`）
- `目标分辨率:` → `Target resolution:`
- 所有提示词精准控制不输出分辨率信息

### 修复
- 错误日志不再静默吞掉：`requests.RequestException` 和 `JSONDecodeError` 会打印到控制台
